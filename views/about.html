ABOUT

The pertinent project attempts to visualize the flow of migrants and asylum-seekers through my home country Slovenia. The visualization was built using D3.js.

THE MAP

The map of Slovenia was generated from a JSON file that contains coordinates of all the European countries. Chapter 12 of Scott Murrayâ€™s book Interactive Data Visualization for The Web greatly facilitated my job of creating a map from a JSON file.

THE DATA

Slovenian police monitor and release information about the influx of migrants on a daily basis. The data is usually released 3 times per day - at 6am, 12am and 6pm - and contains information regarding the number of people that enter and leave the country as well as information on the occupancy of immigration centers across the country. The data is released in the form of MS Word documents, which D3 does not support. I thus had to copy individual values from each word document to first create a spreadsheet and then a .csv file that I used in d3. The spreadsheet contains data on the occupancy of immigration centers for every day from October 20, 2015 (which is when the police started collecting the data) to November 5, 2015(which is when I presented the project in class). The number of people in different immigration centers is represented by differently sized circles. The radius of each circle corresponds to the number of people in a particular center.
